好的习惯不是强迫出来的

发表于 2016-12-19 | 分类于 随笔 | |

靠强迫自己培养出来的好习惯是不能长久的.这是我近几天的深刻感悟.

什么样的最自律 : 我觉得是老板. 为什么?

因为公司赚的钱全部进自己的账户,是多么的刺激和兴奋.有了活马上接,毫不犹豫的安排员工做.所以当一份几百万的合同放到眼前时,你还会说: “等我打完这盘游戏的”吗?

什么样的人最珍惜生命 : 我觉得是老人. 为什么?

老人自己知道已经接近自然规律的界限,自然每天保持养生,运动,乐观的生活.不去计较细小的得失. 而有些年轻人,没有意识到生命的长度.纵酒,吃垃圾食品,每天宅着,睡到中午…大把的挥霍生命.

这样就讲到了自律的本质 :

自律不是外部强加的东西,而是强烈的”我想要”.是自我意识的体现,是一种感到愉快并逐渐习惯的一种生活态度,与身心有益的东西必定是使人舒服,即使开始的时候需要克服一定的阻力.

老板因为想要更多的钱,老人因为想要延年益寿,所以他们能对自己严格的自律.

那么如何才能自律呢 :

我觉得是简单的两个字 :

早起!

欧洲的心理学家发现 : 在早起的人的潜意识里,每天可支配的时间更长.晚起的人,上午时间很短,往往以中午12点为分界线,如果总是很晚才起床,一起床半天就没了,人的心理难免受影响,进而焦虑,同时效率也会相对变的低下,总之坏处多多.

如果晚上23点就入睡,然后早上6.30就起床,这样在上班之前,至少拥有了2小时的自由时间.

当你把一种好的习惯变成自然,他会成为你极其有利的武器,就像007的枪,李小龙的双节棍,海明威的笔.

想要保持自律,就要尽量让自己的日常规律化.每天起床后,把当天的事情简单罗列一下,包括今天要处理的工作,看的书.

罗列以后,一件一件来,不要急.

作为年轻的一员,我希望自己能从自律这里拿到更多的东西.
